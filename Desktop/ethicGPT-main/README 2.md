### 3. OpenAI API
1. Visit [OpenAI Platform](https://platform.openai.com)
2. Create an account and generate an API key
3. Select "OpenAI GPT" in the chatbot
4. Enter your API key when prompted
5. **Note**: Usage costs apply

### 4. Hugging Face API
1. Visit [Hugging Face](https://huggingface.co)
2. Create a free account
3. Go to Settings > Access Tokens
4. Create a new token
5. Select "Hugging Face" in the chatbot
6. Enter your token when prompted
7. **Note**: Free tier with rate limits

### 5. Anthropic Claude API
1. Visit [Anthropic Console](https://console.anthropic.com)
2. Sign up for an account
3. Generate an API key
4. Select "Anthropic Claude" in the chatbot
5. Enter your API key when prompted
6. **Note**: Usage costs apply
